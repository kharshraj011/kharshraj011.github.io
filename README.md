# Sales Dashboard

Interactive sales analytics dashboard designed to monitor business performance through KPI tracking, trend analysis, and product-level insights.

## Live Demo
https://kharshraj011.github.io/

## Project Overview
This dashboard presents sales performance using visual components such as KPI cards, trend charts, monthly comparisons, and product contribution analysis. It is built to transform raw sales data into clear business insights for decision making.

## Features
- Total sales, orders, and average order value KPIs
- Monthly sales trend analysis
- Product-wise sales breakdown
- Filter-based dashboard interaction
- Recent sales transaction tracking
- Business performance summary cards

## Tools & Technologies
- React
- TypeScript
- Tailwind CSS
- Lovable AI

## Business Insights
- Mechanical Keyboard is the highest-selling product
- Desk Lamp shows the lowest sales contribution
- Monthly patterns highlight sales fluctuations

## Purpose
Created as a portfolio project to demonstrate dashboard design, frontend reporting, and business-oriented data presentation skills.
